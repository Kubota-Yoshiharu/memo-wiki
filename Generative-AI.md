
## 用語

| 略称 | 正式名称                   | 和訳             | 意味                                                                   |
| ---- | -------------------------- | ---------------- | ---------------------------------------------------------------------- |
| LLM  | Large Language Models      | 大規模言語モデル | 巨大なデータセットとディープラーニング技術を用いて構築された言語モデル |
| SLM  | Small Language Models      | 小規模言語モデル | 少ない学習データ、小さなパラメータ数でありながら、高性能な言語モデル   |
| VLM  | Vision and Language Models | 大規模視覚モデル | 画像とテキストを複合して扱うモデル全般                                 |
|      |                            |                  |                                                                        |
